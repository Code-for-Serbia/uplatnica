# Uplatnica

Uplatnica is project for Serbian PaySlip, vision is to create easiest way to generate Serbian payslip and NBS QR code.

# Development

## Setup

Clone

```bash
git clone https://github.com/itmilos/uplatnica.git
cd uplatnica
```

Install dependencies

```bash
npm i -f
```

## Start

To start the app:

```bash
npm run dev
```
